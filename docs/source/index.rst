Rhoban édition Education
========================

Le projet Rhoban est principalement constitué d'un :ref:`serveur <server>`, 
qui accède à une  couche bas niveau constituée de moteurs, de capteurs etc. 
et permet de les manipuler de manière plus abstraite.

La couche logicielle fournit également un système de conception de mouvements
que l'on peut concevoir à l'aide de l':ref:`IHM <ihm>`.

Il est aussi possible d'utiliser le :ref:`SDK <sdk>` pour écrire votre propre
code qui intéragira avec le serveur et l'ensemble du matériel qui y est connecté.
